# AI Resume Screening System

An ATS-inspired Resume Screening System built using Python, Streamlit, and Sentence Transformers.

# Features

* Upload multiple PDF resumes
* Upload Job Description (JD)
* Skill matching and missing skill detection
* AI semantic similarity scoring
* Candidate ranking and best candidate selection
* Email and mobile number extraction
* Interactive summary table and CSV export

# Example Output

* Best Candidate Selection
* Candidate Ranking
* Matched Skills
* Missing Skills
* Email Address
* Mobile Number
* Final ATS Score

# Tech Stack

* Python
* Streamlit
* Pandas
* PyMuPDF
* Sentence Transformers
